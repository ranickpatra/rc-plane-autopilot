# Mono Propeller Drone

## connections

| PIN     | PORT    | USED          |
| :---    | :---    | :---          |
| D8      | PB0     | Fin 1         |
| D9      | PB1     | Fin 2         |
| D11     | PB3     | Fin 3         |
| D12     | PB4     | Fin 4         |
| D10     | PB2     | Propeller     |

## Receiver Connection

| PIN       | PORT  | PCINT     | USED |
| :---      | :---  | :---      | :--- |
| D4        | PD4   | PCINT20   | CH1  |
| D5        | PD5   | PCINT21   | CH2  |
| D6        | PD6   | PCINT22   | CH3  |
| D7        | PD7   | PCINT23   | CH4  |
